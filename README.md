# 🏃 Heist Run — Endless Thief Runner

Subway Surfers–style 3D endless runner। চোর হিসেবে তিন লেনে দৌড়াও, কয়েন সংগ্রহ করো, পুলিশ এড়িয়ে চলো!

---

## 🎮 কীভাবে খেলবেন

| কন্ট্রোল | কীবোর্ড | মোবাইল |
|----------|---------|--------|
| বাম লেন | `←` / `A` | Swipe বাম / ◀ বাটন |
| ডান লেন | `→` / `D` | Swipe ডান / ▶ বাটন |
| লাফ | `Space` / `↑` / `W` | Swipe উপর / ▲ বাটন |
| ড্যাশ | `Shift` | Tap স্ক্রিন / ⚡ বাটন |
| পজ | `P` / `Esc` | — |
| শপ | `S` (মেনুতে) | — |
| আবার খেলা | `R` (গেম ওভারে) | Run Again বাটন |

## 💾 সেভ সিস্টেম (Subway Surfers স্টাইল)

গেম **localStorage**-এ অটো সেভ হয় (`heistRunSave_v2`):

| ডেটা | বর্ণনা |
|-------|--------|
| `totalCoins` | সব রানের মোট কয়েন (স্থায়ী) |
| `highScore` | সর্বোচ্চ স্কোর |
| `bestLevel` | সর্বোচ্চ পৌঁছানো লেভেল |
| `totalRuns` | মোট কতবার খেলেছেন |
| `dailyBest` | আজকের সেরা স্কোর |
| `upgrades` | কেনা আপগ্রেড (Dash Boost, Lucky Start) |

### Continue (Revive)
গেম ওভারে **🪙 50 কয়েন** খরচ করে একবার চালিয়ে যেতে পারবেন (প্রতি রানে একবার)।

---

## 🗺️ লেভেল সিস্টেম

প্রতি **~১০ সেকেন্ড**ে লেভেল বাড়ে (সর্বোচ্চ ১০০)। প্রতি লেভেলে গতি সামান্য বাড়ে।

| লেভেল | নাম | নতুন কী আসে |
|-------|-----|-------------|
| **1** | Street Heist | কয়েন সংগ্রহ — রাস্তার পাশে গাছের কাঠ |
| **2** | Riverside Run | নতুন ব্যাকগ্রাউন্ড — পরিষ্কার দৃশ্য |
| **3** | Metro Escape | ট্রেন — লেন বদলাতে হবে! |
| **5** | Double Trouble | একসাথে দুই বাধা |
| **10** | Sky Loot | বেশি এয়ার কয়েন + কম্বো বোনাস |
| **20** | Night Raid | দ্রুত স্পন + ডাবল বাধা |
| **50** | Legend Heist | সর্বোচ্চ চ্যালেঞ্জ |

লেভেল আপ হলে স্ক্রিনে নাম ও নতুন ফিচার দেখায়।

## ✨ নতুন ফিচার ও ফিক্স (আপডেট লিস্ট)

### গেমপ্লে
- ✅ **baseSpeed / gameSpeed** আলাদা — ড্যাশ শেষে স্পিড লিক হয় না
- ✅ **কম্বো সিস্টেম** — পরপর কয়েনে বোনাস স্কোর
- ✅ **স্মার্ট স্পন** — একই লেনে বারবার বাধা কম
- ✅ **ড্যাশ ম্যাগনেট** — ড্যাশে কাছের কয়েন আকর্ষিত হয়
- ✅ **Revive** — কয়েন দিয়ে একবার বাঁচা
- ✅ **Invincibility** — Revive পর স্বল্প সময় অজেয়

### UI / Polish
- ✅ মেইন মেনু (স্কোর, কয়েন, বেস্ট দেখায়)
- ✅ লেভেল আপ ব্যানার
- ✅ গেম ওভার প্যানেল (HTML overlay)
- ✅ পজ স্ক্রিন
- ✅ কয়েন পার্টিকেল + ফ্লোটিং +স্কোর টেক্সট
- ✅ স্ক্রিন শেক যখন ধরা পড়েন
- ✅ ড্যাশে স্পিড লাইন ইফেক্ট
- ✅ মোবাইল টাচ বাটন + সোয়াইপ

### টেকনিক্যাল
- ✅ কোড `game.js`-এ আলাদা (পরিষ্কার স্ট্রাকচার)
- ✅ `localStorage` সেভ/লোড
- ✅ মোবাইল viewport + `touch-action` অপটিমাইজ
- ✅ ইমেজ/সাউন্ড না থাকলে রঙিন ফলব্যাক

## 📁 ফাইল স্ট্রাকচার

```
heist-run/
├── index.html      # গেম শেল + CSS + টাচ বাটন
├── game.js         # পুরো গেম ইঞ্জিন
├── README.md       # এই ফাইল
├── images/         # স্প্রাইট (thief, police, car, train, bg...)
│   ├── thief.png
│   ├── police.png
│   ├── moneybag.png
│   ├── log_transparent.png
│   ├── car.png
│   ├── train.png
│   ├── background_level1.png
│   ├── background_level2.png
│   └── background_level3.png
└── sounds/
    ├── bg-music.mp3   ← মেনু/গেম মিউজিক (না থাকলে অটো fallback মিউজিক)
    ├── coin.mp3
    ├── jump.mp3
    └── busted.mp3
```

> **নোট:** `bg-music.mp3` যোগ করলে আসল মিউজিক বাজবে। না থাকলেও গেম **Web Audio fallback** মিউজিক চালায়। SFX-ও একইভাবে fallback বীপ ব্যবহার করে।

## 🚀 চালানোর উপায়

1. ফোল্ডারে `images/` ও `sounds/` যোগ করুন
2. `index.html` ব্রাউজারে খুলুন
3. অথবা লোকাল সার্ভার:
   ```bash
   npx serve .
   ```

## 🎯 টিপস

- **ট্রেন**-এ লাফ কাজ করে না — লেন বদলান
- **লগ/গাড়ি**-তে লাফ দিয়ে এড়ানো যায়

- কয়েন জমিয়ে **Revive** ও **আপগ্রেড** কিনুন
